# Árbol Binario de Búsqueda – Java

Trabajo práctico individual realizado para la materia **Algoritmos y Estructuras de Datos** (UBA).

El objetivo del trabajo es implementar un conjunto genérico utilizando un árbol binario de búsqueda.

---

## Descripción

Se implementa una estructura de datos que permite:

- Insertar y eliminar elementos
- Consultar pertenencia
- Obtener el mínimo y el máximo
- Recorrer los elementos en orden mediante un iterador

La implementación respeta las propiedades de los árboles binarios de búsqueda y las interfaces provistas por la cátedra.
